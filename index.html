FinTech Global Rebrand

創業50年を迎える金融機関のブランディング刷新に伴い、グローバルWebサイトのリニューアルを担当しました。信頼性を損なわずに、デジタル時代に即した先進的なイメージを表現することが求められました。

Role

Lead UI Designer
Frontend Developer

Tech Stack

Figma, Astro, Tailwind CSS

Challenge & Solution

旧サイトは情報構造が複雑で、モバイル対応も不十分でした。1,000ページを超えるコンテンツを整理し、アトミックデザインに基づいたコンポーネント設計を導入することで、運用効率とUIの一貫性を劇的に向上させました。

Key Results

ページ読み込み速度 40% 向上
問い合わせコンバージョン 15% 増加

Lifestyle Store App

ユーザーのライフスタイルに寄り添う、没入感のあるショッピング体験を目指しました。写真の魅力を最大限に引き出すため、UI要素は極力排除し、ジェスチャー主体の操作系を採用しました。

Role

UI/UX Design
Prototyping

Tech Stack

Figma, Swift UI

Challenge & Solution

商品検索から購入までのステップ数を削減することが課題でした。ユーザーテストを繰り返し、直感的なフィルタリング機能と「あとで買う」リストへのスムーズな動線を設計しました。
